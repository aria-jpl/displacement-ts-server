Interferogram Timeseries Visualization

Timeseries of

This tool enables the user to explore time-series products archived by the ARIA project by visualizing them using Leaflet Maps. It also enables the user to track explicit locations on the map by dropping markers, and graphing the data under that marker. Simply double-click on the location(s) desired.

Instructions:

Changing layers

  1. Click on
  2. Select desired layers from dropdown

Adding Markers

  1. Navigate using map to find desire location
  2. Double-click to add marker
  3. Set desired interval on chart

Clear Markers

  1. Click on
  2. Deselect and re-select layer to clear markers
